## 🔄 Data Flow

### 1. **User Authentication (Native Token Exchange)**
Native Auth (Apple/Google) → `native-auth-exchange` Cloud Function → Appwrite Session.

### 2. **Offline-First Data Sync**
UI Action → GRDB (Local DB) → `SyncQueue` → `OfflineSyncEngine` → Appwrite Collections.
FCM "sync_nudge" triggers background selective pulls.

### 3. **AI Meal Verification & Quota**
Camera/Image → Gemini Vision → `AIQuotaManager` checks limits → Log Meal.

### 4. **Progressive Plan Generation**
User Input (Wizard) → `NewMealPlanWizardViewModel` → AI Analysis → `ProgressivePlan` stored locally & synced.
